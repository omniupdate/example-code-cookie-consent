Cookie Consent Sample

This is a sample of using the Osano cookie consent library to set up a cookie consent bar for your website as seen at the bottom of this page. It is licensed under MIT, so it is free for personal and commercial use.

An easy way to configure the position, design, and content of the consent bar is actually using the osano site's configure form. Visit their download page and click on the "Start Coding" button to open up their configuring form and follow the direction on the page.

The code used to enable the cookie bar is shown below.

Place this inside headcode.inc (usually located at /_resources/includes/headcode.inc):

<link rel="stylesheet" type="text/css" href="https://cdn.jsdelivr.net/npm/cookieconsent@3/build/cookieconsent.min.css" />

Place this inside footcode.inc (usually located at /_resources/includes/footcode.inc):

<script src="https://cdn.jsdelivr.net/npm/cookieconsent@3/build/cookieconsent.min.js" data-cfasync="false">
<script>
    window.cookieconsent.initialise({
        "palette": {
            "popup": {
                "background": "#64386b",
                "text": "#ffcdfd"
            },
            "button": {
                "background": "#f8a8ff",
                "text": "#3f0045"
            }
        },
        "theme": "edgeless"
    });
</script>
